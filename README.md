# Box_For_Ksu
专为 KernelSU 打造的一站式配置仓库，集成 mihomo、内核、模块等实用配置，开箱即用。

## ✨ 项目简介
本仓库用于存放和维护基于 **KernelSU** 的各类配置文件，
涵盖网络代理、内核参数、模块配置等，方便快速部署与同步。

## 📦 包含内容
- **mihomo 配置**
  - 通用稳定配置
  - 规则组、代理组、TUN 等优化参数
- **KernelSU 内核配置**
  - 性能/功耗平衡参数
  - 启动项与调度优化
  - **KSU 模块配置**
  - 常用模块预设
  - 权限、自启动配置模板
- **辅助脚本与说明**
  - 配置备份、替换脚本
  - 常见问题排查

## 🚀 使用方式
  1. 克隆或下载本仓库
  2. 将对应配置文件放入设备对应路径
  3. 按需微调后重启服务/模块生效

## 🔧 适用人群
  - KernelSU 用户
  - mihomo / 透明代理使用者
  - 喜欢自定义、折腾配置的玩家

## 📌 特点
  - 配置干净、注释清晰
  - 持续维护，兼容新版 KSU & mihomo
  - 结构规范，便于同步与二次修改
